Follow the instructions below to get started.

1. Navigate to the 'server' folder and run 'npm install mongodb'
2. Go to the 'data-import' folder and run 'node mongodb-import.js'
3. run 'node combineddb.js' to get a database 'movieDb'
4. Navigate back to 'server' folder and run 'server.js'

Now you should have a database called 'movieDb' and be connected to the server.
